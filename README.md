# Crypto-analysis-powerbi
## Description
Projet de Data Visualisation réalisé dans le cadre du module Data Visualisation à l'INSEA.
Il consiste en une analyse approfondie de 7 cryptomonnaies majeures à partir de données 
historiques récupérées via Yahoo Finance, visualisées sous Power BI Desktop.
## Cryptomonnaies analysées
| Ticker | Nom |
|--------|-----|
| BTC-USD | Bitcoin |
| ETH-USD | Ethereum |
| BNB-USD | Binance Coin |
| SOL-USD | Solana |
| ADA-USD | Cardano |
| DOGE-USD | Dogecoin |
| XRP-USD | Ripple |
## Technologies utilisées
- **Python** (Google Colab) — collecte et traitement des données
- **yfinance** — extraction des données Yahoo Finance
- **pandas / numpy** — manipulation des données
- **Power BI Desktop** — visualisation interactive
## Principaux résultats
- **Bitcoin (BTC)** offre le meilleur rapport rendement/risque
- **Solana (SOL)** et **Ethereum (ETH)** sont les plus volatils
- **ADA, DOGE, XRP** présentent des performances plus limitées
- Le marché suit des cycles haussiers/baissiers clairement identifiables
- Forte corrélation entre les cryptos lors des phases de stress
